# Invoice Cost Analytics & Decision Support System

## Project Overview
This project delivers a comprehensive invoice cost analytics solution designed to improve financial visibility, cost governance, and data-driven decision-making across property portfolios and geographic regions.

The solution transforms raw invoice data into structured, reliable insights that help stakeholders understand where money is being spent, identify key cost drivers, and prioritise areas for cost optimisation.

---

## Business Problem
Organisations managing large property portfolios often struggle with:
- Limited visibility into cost distribution
- Difficulty identifying high-cost locations and asset types
- Lack of clarity around expense drivers
- Manual and inconsistent reporting processes

This project addresses these challenges by centralising invoice data and enabling consistent analysis across financial, geographic, and asset dimensions. :contentReference[oaicite:0]{index=0}

## Data Scope
The dataset includes:
- Invoice cost values
- Property types
- Suburbs and postcodes
- Local Government Authorities
- State and federal electorates
- Expense categories (Maintenance, Modifications)
